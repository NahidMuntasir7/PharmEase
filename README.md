# üíä PharmEase

**PharmEase** is a Java-based pharmacy management system developed during my third semester. The project applies Object-Oriented Programming (OOP) principles, utilizes **Java Swing** for the graphical user interface (GUI), and integrates **MySQL** for efficient database management.

## üìú Project Overview

PharmEase is designed to streamline pharmacy operations, providing a comprehensive solution for managing inventory, sales, and customer relations. The application aims to enhance productivity and efficiency in a pharmacy setting by offering a suite of features that address common management challenges. 

With its user-friendly interface, PharmEase allows users to easily navigate through various functionalities, whether they are administrators managing user accounts or pharmacists handling medicine sales. The system incorporates essential features like inventory tracking, which enables users to monitor stock levels, receive alerts for low inventory, and manage medicine details efficiently.

The application also supports user management, allowing administrators to create, update, and delete user accounts, ensuring that access is controlled and monitored. By utilizing MySQL for database management, PharmEase ensures data integrity and security while providing quick access to necessary information.

This project exemplifies the practical application of OOP concepts, showcasing encapsulation, inheritance, polymorphism, and abstraction to create a modular and scalable system. Whether you're managing customer profiles or generating sales reports, PharmEase is built to support the daily operations of a pharmacy with ease and reliability.

### Key Features:

- **User-Friendly Interface**: The application leverages **Java Swing** to provide an intuitive, easy-to-navigate graphical user interface (GUI).
  
- **Database Integration**: MySQL is used for effective storage and management of all pharmacy-related data, ensuring smooth and reliable data processing.
  
- **OOP Concepts**: The system showcases core OOP concepts such as encapsulation, inheritance, polymorphism, and abstraction, making the code maintainable and scalable.
